层叠样式表:
CSS:Cascahe StyleSheet.
根据样式代码的位置：样式表分为三类:
1.行内样式:一个页面中某个标签不同于同类标签的外观：行内样式使用元素style属性定义:
   语法:<p style="样式一:值一;样式二:值二;..."></p> 注意分号一定要加
  
2.内嵌样式:行内样式表局限于某个标签，如果希望本网页内的所有同类标签都采用统一样式，这时应采用内嵌样式。(页面中所有的同类标签呈现同样的效果)
语法:<style type="text/css">内嵌样式 </style>
 三种内嵌样式:
1)HTML选择器:使用HTML标签来标识的
  语法:
  标签
  {
      样式一:值一;
      样式二:值二;
      ...
  }
  注意:当定义了HTML选择器样式后会自动应用到页面中所有同类标签
2)Class类选择器:
  在Style标签内部定义:
  语法:
   .类名
   {
      样式一:值一;
      样式二:值二;
      ...
   }
   应用class类选择样式:<p class="类名"></p>
3)ID选择器样式:
  在Style标签内部定义:
  语法:
   #ID名
   {
      样式一:值一;
      样式二:值二;
      ...
   }
   应用ID选择器样式:<p ID="ID名"></p>
4)伪类链接:适用于所有超链接
   a{样式}
   a:hover{样式}:鼠标悬停到超链接时的效果

3.外部样式 :用于同一站点中的所有的页面(达到同一站点的页面外观样式统一)
  应用外部样式步聚:1)创建外部样式文件
                    2)引入到要使用外部样式的页面(引入方法有两种)

引入方式: 方式一:链接(link)外部样式表   方式二:导入外部样式表:import关键字

对于某个HTML标签：
1）如果有多种样式，如果规定的样式没有冲突，则叠加；
2）如果有冲突，则最先考虑行内样式表显示，如果没有，再考虑内嵌样式显示，如果还没有，最后采用外面样式表显示，否则就按HTML的默认样式显示；
